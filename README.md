# 🌦️ Weather Web Application

A responsive Weather Website built using HTML, CSS, 
This project displays different weather conditions with interactive UI pages.

---

## 🚀 Features

- 🌤️ Sunny Weather Page
- 🌧️ Rainy Weather Page
- ☁️ Cloud Weather Page
- ⛈️ Storm Weather Page
- 🔍 Explore Weather Section
- Responsive Design (Desktop Optimized)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
